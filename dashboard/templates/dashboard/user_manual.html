Creating Articles

Creating a new article is straightforward with our enhanced editor.

Step-by-Step Guide

  1. Navigate to Article Creation

    Click "Create Article" in the sidebar or use the quick action button on the dashboard.

  2. Enter Article Title

    Type your article title in the large input field at the top. This will automatically generate the URL slug.

  3. Set Publication Date

    Choose when to publish your article using the date/time picker.

  4. Select Category and Author

    Choose from existing categories and authors, or create new ones using the "+" buttons.

  5. Write Article Excerpt

    Add a compelling summary (up to 500 characters) that will appear in article previews.

  6. Create Content

    Use the rich text editor to write your article with formatting, images, and special elements.

  7. Set Featured Image

    Upload and select a featured image from the media library in the sidebar.

  8. Configure SEO Settings

    Add meta title, description, and keywords for search engine optimization.

  9. Add Tags

    Include relevant tags to help categorize and organize your content.

  10. Save and Publish

    Save as draft or publish immediately. The system auto-saves every 30 seconds.

Important: Always fill out the title, excerpt, category, and author fields - these are required for publishing.

Editing Articles

Edit existing articles with full access to all creation features plus additional editing tools.

Accessing Articles for Editing

  • From the Dashboard: Click the edit icon next to any recent article
  • From Articles List: Navigate to "All Articles" and click the edit button
  • From Article View: Click "Edit Article" when viewing a published article

Media Management

Upload, organize, and insert media files with automatic optimization.

Uploading Media

  1. Access Media Library

    Click "Media Library" in the sidebar or use the image button in the article editor.

  2. Upload Files

    Click "Upload Files" button or drag and drop files directly onto the page.

  3. Automatic Processing

    Images are automatically optimized to WebP format and resized for web performance.

Supported File Types

Type Formats Max Size Notes
Images JPG, PNG, GIF, WebP 10MB Auto-optimized to WebP
Videos MP4, WebM, OGG 100MB Use for short clips
Documents PDF, DOC, DOCX, TXT 25MB For downloadable resources

Inserting Images into Articles

  1. Position Cursor

    Click where you want to insert the image in your article.

  2. Open Media Library

    Click the image icon in the toolbar or use the Insert menu.

  3. Select Image

    Click on an image to select it, then click "Insert Selected".

  4. Add Caption

    Click below the inserted image to add an optional caption.

Auto-Optimization: All images are automatically compressed and converted to WebP format for faster loading times.

Categories & Authors

Organize your content with categories and manage author information.

Creating Categories

  1. Access Category Creation

    Click the "+" button next to the category dropdown when creating/editing articles.

  2. Enter Category Details

    Provide a name (URL-friendly), display name, description, and color.

  3. Save and Use

    The new category becomes immediately available for selection.

Managing Authors

  1. Create New Author

    Use the "+" button next to the author dropdown.

  2. Author Information

    Enter name, professional title, email, and biography.

  3. Author Profiles

    Authors can have detailed profiles with photos and social links.

Best Practice: Create author profiles for all regular contributors to maintain consistency and professionalism.

SEO Settings

Optimize your articles for search engines with comprehensive SEO tools.

SEO Checklist

Element Recommended Length Purpose
Meta Title 50-60 characters Appears in search results and browser tabs
Meta Description 150-160 characters Snippet shown in search results
Focus Keywords 3-5 keywords Main terms you want to rank for
Alt Text (Images) 125 characters max Accessibility and image SEO

SEO Best Practices

  • Use your focus keyword in the title and first paragraph
  • Write compelling meta descriptions that encourage clicks
  • Use header tags (H1, H2, H3) to structure content
  • Add alt text to all images
  • Keep URLs short and descriptive
  • Internal linking to related articles

Publishing Workflow

Understand the publishing process and content lifecycle.

Article Status Stages

Draft

Work in progress, visible only to editors

Under Review

Submitted for editorial review

Scheduled

Set to publish at a specific date/time

Published

Live on the website and visible to visitors

Pre-Publication Checklist

  1. Content Review

    Check spelling, grammar, and factual accuracy.

  2. Image Verification

    Ensure all images have alt text and appropriate captions.

  3. SEO Optimization

    Complete meta title, description, and focus keywords.

  4. Preview Article

    Use the preview function to see how it will appear to readers.

  5. Final Formatting

    Check heading hierarchy and overall layout.

Troubleshooting

Solutions to common issues and how to get help.

Common Issues

Editor Not Loading
  • Refresh the page (F5 or Ctrl+R)
  • Clear browser cache and cookies
  • Try a different browser
  • Check internet connection
Auto-Save Not Working
  • Check if you're still logged in
  • Verify internet connection
  • Look for error messages in the browser console
  • Manual save with Ctrl+S
Images Not Uploading
  • Check file size (max 10MB for images)
  • Verify file format is supported
  • Try uploading one file at a time
  • Check internet connection stability
Formatting Issues
  • Use the "Clear Formatting" button (Ctrl+Space)
  • Copy content to plain text editor and back
  • Check for conflicting CSS styles
  • Refresh the editor
Need Help? If you encounter persistent issues, contact your system administrator or technical support team with specific details about the problem and steps you've tried.
